# Add waves

Add waves

## Usage

``` r
add_wave(
  .x,
  date_var,
  normals_df,
  threshold = 5,
  threshold_cond = "gte",
  size = 3,
  var_name = "hw3"
)
```

## Arguments

- .x:

  data

- date_var:

  date var

- normals_df:

  normal df

- threshold:

  threshold

- threshold_cond:

  threshold_cond

- size:

  size

- var_name:

  var_name

## Value

data frame
